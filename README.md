# SpamShield

**SpamShield** is a lightweight Minecraft plugin that prevents repeated spam messages in chat.  
It blocks identical messages or repeated words from appearing, keeping chat clean and readable without restricting gameplay.

## Features

- Blocks repeated messages (e.g., "when when when")  
- Simple, lightweight, and no external dependencies  
- Fully configurable via `config.yml` (future updates)  
- Works on Paper/Spigot 1.21+  

## Installation

1. Download `SpamShield.jar`  
2. Place it in your server's `plugins` folder  
3. Start or restart the server
